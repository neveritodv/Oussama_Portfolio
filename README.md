# Oussama Moujane — Full Stack & Mobile Developer Portfolio

A modern, high-performance developer portfolio for **Oussama Moujane (Oussama Dev)**, showcasing full-stack web platforms, mobile applications (Flutter / React Native), SaaS systems, and API backend architectures.

Built with **Next.js 15 App Router**, **React 19**, **TypeScript**, **Tailwind CSS v4**, and smooth physics animations powered by **Motion**.

---

## ✨ Features

- **⚡ Hero Showcase**: High-impact landing layout featuring Oussama Dev's custom visual branding, developer profile, interactive contact triggers, and quick stats.
- **📁 Comprehensive Project Gallery**: Interactive showcase featuring live preview cards, categorized technology tags, and modal deep-dives.
- **🖼️ Multi-Column Case Studies**: Rich project modal detailing architectural highlights, technical stack badges, repository/demo links, and multi-image galleries.
- **🛠️ Interactive Tech Stack & Marquee**: Visual overview of core skillsets across Front-end, Back-end, Mobile Development, Databases, and DevOps tooling.
- **⏳ Professional Journey**: Animated experience timeline tracing education at *École Racine Marrakech* and key software engineering milestones.
- **🎨 Dark Theme Aesthetic**: Custom-crafted dark UI with glow accents, glassmorphic cards, and high contrast typography.
- **📱 Fully Responsive**: Fluid layouts engineered for desktop, tablet, and mobile displays.

## 🚀 Getting Started

### Prerequisites

Ensure you have [Node.js](https://nodejs.org/) (v18.0 or later) installed on your system.

### Installation

1. **Clone or download the repository**:
   ```bash
   git clone https://github.com/moujane1023/portfolio.git
   cd portfolio
   ```

2. **Install dependencies**:
   ```bash
   npm install
   ```

3. **Start the development server**:
   ```bash
   npm run dev
   ```

4. Open your browser and navigate to `http://localhost:3000`.

---

## 📜 Available Scripts

- `npm run dev` — Runs the application in development mode with hot reloading.
- `npm run build` — Compiles the optimized production build.
- `npm run start` — Starts the Next.js production server.
- `npm run lint` — Runs ESLint checks across the codebase.
